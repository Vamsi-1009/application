Master Cybersecurity Checks & Resources

A consolidated HTML reference from our conversation: application, API, mobile, AI, code, vulnerabilities, server hardening, cloud, containers, supply chain, identity, threat intelligence, penetration testing, IoT, OT, automotive, hardware, compliance and governance.

How to use this: These resources are not all automated scanners. Some are testing methodologies, some are standards/checklists, some are vulnerability intelligence feeds, and some are compliance or hardening frameworks. Apply only the areas relevant to the system you own or are authorized to assess.
